# LARP百科事典 (LARPencyclopedia)

クリエイターユニット**CLOSS（雛咲・星屑）**による、LARP運営とプレイの「唯一の正典（Single Source of Truth）」となる高解像度知識ベース。

## 📖 概要
本プロジェクトは、LARP（Live Action Role Playing）というメディアが持つ、身体性、即興性、および心理的安全性の課題を解決するための高度な知恵を体系化したものです。
単なる用語集ではなく、**「Play to Contribute (PtC)」** という共同創造のOSを核とした、プロフェッショナルな運営基準を定義しています。

## 📁 ディレクトリ構造
- `data/dictionary_master.md`: 全用語のインデックス。カテゴリ別（逆引き）と五十音順で整理されています。
- `data/terms/*.md`: 各用語の詳細な定義、運用、実例、および注意事項。

## 🚀 主要なプレイ哲学
- **Play to Contribute (PtC)**: 個人の勝利を超え、物語全体の質に貢献する。
- **Play to Win / Lose / Lift**: 勝利、敗北、他者の扶助を、物語を彩るための戦術的リソースとして使い分ける。

## 🛡️ 安全へのコミットメント
CLOSSのLARPは、以下のツールを標準装備することで、物理的・精神的安全を担保します。
- **ベールとライン**: 描写の境界線の設計。
- **キャリブレーション**: プレイ強度の同期。
- **タップアウト / オプトアウト**: プレイヤーの拒否権の絶対的尊重。
- **OKチェック / チェックイン**: リアルタイムの安全確認。

## 🛠️ 使い方
1. [辞書マスター](./data/dictionary_master.md) を開き、必要なカテゴリから用語を探します。
2. 各用語の「関連用語」リンクを辿ることで、関連する知恵を芋づる式に習得できます。
3. GM、PC、NPC、主催者それぞれの視点からの「詳細定義」を参照し、現場での指針として活用してください。
